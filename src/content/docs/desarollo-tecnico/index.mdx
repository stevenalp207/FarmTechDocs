Bienvenido a la documentación técnica del **FT-01**, un robot modular agrícola diseñado para la adquisición de datos terrestres y satelitales, análisis autónomo mediante Inteligencia Artificial y generación de reportes automáticos para la toma de decisiones agrícolas inteligentes.

Esta sección proporciona una visión general del sistema y enlaza a las distintas áreas técnicas que componen el desarrollo del FT-01.

- [Arquitectura General del Sistema](desarollo-tecnico/arquitectura): Visión modular y comunicación entre los diferentes componentes del robot.
- [Módulo de Adquisición de Datos Terrestres](desarollo-tecnico/modulos-adquisicion): Detalles técnicos de los sensores BMP390, SCD30, LTR390 y de humedad de suelo.
- [Módulo de Adquisición de Datos Satelitales](desarollo-tecnico/modulos-adquisicion): Descripción del GPS HAT y su integración para posicionamiento preciso.
- [Procesamiento y Análisis con IA](desarollo-tecnico/ia): Uso de Jetson Nano con Ollama y Gemma3 para generación de reportes.
- [Alimentación, Comunicación y Movilidad](desarollo-tecnico/energia-comunicacion): Diseño del sistema de potencia, motores NEMA 17 y comunicación entre placas.
- [Sistema de Logs](desarollo-tecnico/logs): Registro y almacenamiento de datos operativos mediante Python `logging`.
