Esta sección expone los principales resultados obtenidos durante el desarrollo y las pruebas del robot **FT-01**, diseñado para ayudar a los agricultores en la toma de decisiones sobre qué producto sembrar, considerando las condiciones específicas del terreno. El prototipo, equipado con sensores terrestres y conexión a datos satelitales, recopila información ambiental clave que es procesada por una inteligencia artificial (IA) para generar recomendaciones personalizadas.

## Resultados de mediciones terrestres

Para la evaluación del terreno se emplearon sensores especializados conectados al sistema robótico, encargados de medir parámetros ambientales clave. Estos datos permiten identificar qué cultivos pueden desarrollarse con mayor éxito en un área específica.

## Datos satelitales obtenidos

Para complementar los datos terrestres, se utilizó la API **NASA POWER** (Prediction Of Worldwide Energy Resources) que proporciona datos meteorológicos históricos y actuales a partir de coordenadas geográficas.

Se empleó un script en Python para realizar solicitudes programáticas de variables climáticas clave, como:

* Temperatura media y extrema
* Humedad relativa
* Precipitación
* Radiación solar
* Velocidad del viento
* Punto de rocío

Las consultas abarcaron un rango de 30 días desde el **10 de mayo** hasta la fecha actual.
